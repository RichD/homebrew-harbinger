# Homebrew Harbinger

Homebrew tap for [stackharbinger](https://stackharbinger.com) - a tool to track End-of-Life dates for your tech stack.

## Installation

First, tap this repository:

```bash
brew tap richd/harbinger
```

Then install stackharbinger:

```bash
brew install stackharbinger
```

## Usage

After installation, the `harbinger` command will be available:

```bash
harbinger version
```

For more information about using stackharbinger, visit [stackharbinger.com](https://stackharbinger.com).

## Updating

To update to the latest version:

```bash
brew update
brew upgrade stackharbinger
```

## Uninstalling

```bash
brew uninstall stackharbinger
brew untap richd/harbinger
```
